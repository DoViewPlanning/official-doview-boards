# DoView Collection Index Prompt Package

This folder contains reusable developer templates for creating:

- a DoView Board collection page; and
- a DoView Board collection-of-collections page.

These files are adapted for the DoView Boards V1.3.7 release.

These files are developer resources, not runtime dependencies for generated DoView Board HTML files.

## Files

- `DoView-collection-upload-prompt.md` - prompt to give to an AI assistant.
- `templates/00-collection-of-collections-index-template.html` - collection-of-collections index template for `/boards/index.html`.
- `templates/01-simple-live-boards-index-template.html` - use when all boxes are live boards.
- `templates/02-top-board-plus-live-boards-index-template.html` - use when there is a centered Top DoView Board and all lower boxes are live boards.
- `templates/03-live-and-placeholder-boards-index-template.html` - use when there are live boards and gray placeholder boxes.
- `templates/01-simple-live-boards-collection-template.json` - matching subfolder `collection.json` template for option 1.
- `templates/02-top-board-plus-live-boards-collection-template.json` - matching subfolder `collection.json` template for option 2.
- `templates/03-live-and-placeholder-boards-collection-template.json` - matching subfolder `collection.json` template for option 3.
- `templates/collection-of-collections-template.json` - minimal structure for the top-level `/boards/collections.json` file.

## How the templates relate

The collection-of-collections page lives at `/boards/index.html`, uses `templates/00-collection-of-collections-index-template.html`, and loads `collections.json`.

Each subfolder collection page lives at `/boards/<folder-name>/index.html`, uses one of the three subfolder index templates, and loads `collection.json`.

Use `collections.json` for the main list of collections. Use `collection.json` inside each subfolder for that collection's boards.

```text
Collection of collections:
  /boards/index.html
  /boards/collections.json

Single collection:
  /boards/<folder-name>/index.html
  /boards/<folder-name>/collection.json
```

## Template use

Public reusable collection templates are badge-free by default. The top-right acknowledgment text is fixed and should normally not be edited.

Developers and users should edit only the title, subtitle, maintainer/publisher line, description/about text, board entries, collection entries, counts, and links. Preserve the card visual design and JSON-loading behaviour.

Individual board HTML files may still carry their own board-level Official DoView Badge if they were generated by the official prompt/package. Collection-level official badges are reserved for official curated DoView collections, not public reusable templates.

## Important style rule

Only the live-plus-placeholder template should display the instruction strip:

`White boxes link to live DoView Boards. Gray boxes are illustrative placeholders.`

Use US spelling throughout.
